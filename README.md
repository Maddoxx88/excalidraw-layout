# Excalidraw Layouts
A collection of templates which you can drag and drop using [Excalidraw](https://excalidraw.com/).

## How to use 

- Download [Mobile Kit](mobile-kit.excalidrawlib).
- Open [Excalidraw](https://excalidraw.com).
- Open library (or press "9").
