# Excalidraw Layouts
A collection of templates/layouts which you can drag and drop into [Excalidraw](https://excalidraw.com/).

## How to use 

- Download a layout of your choice.
- Open [Excalidraw](https://excalidraw.com).
- Open library (or press "9").
